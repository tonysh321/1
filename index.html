用户注册系统 - 直接访问方案

方法1：直接打开API注册页面

点击下方按钮直接访问注册API页面：

直接打开注册页面

方法2：手动构建URL

复制以下URL，替换其中的参数：

http://a.buhui.ltd/main/backend/user/Register.php?
admin=3855584220&
user=你的账号&
password=你的密码&
name=你的昵称&
code=验证码（可选）&
invite_code=邀请码（可选）&
key=公钥（可选）

然后在浏览器地址栏访问即可。

方法3：使用cURL命令

在命令行中执行：

curl -X POST "http://a.buhui.ltd/main/backend/user/Register.php" \
-d "admin=3855584220" \
-d "user=你的账号" \
-d "password=你的密码" \
-d "name=你的昵称" \
-d "code=验证码" \
-d "invite_code=邀请码" \
-d "key=公钥"
为什么需要直接访问？
由于API不支持HTTPS和CORS，现代浏览器会阻止从HTTPS页面访问HTTP资源。直接访问是最可靠的方案。
